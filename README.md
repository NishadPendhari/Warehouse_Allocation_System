# Warehouse_Allocation_System


## Features

- Warehouse Management
- Product Management
- Inventory Tracking
- Stock Allocation
- Stock Transfer
- Audit Logging
- Swagger Documentation
- Exception Handling

## Technologies

- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- Maven
- Swagger OpenAPI

## Run

1. Create MySQL database
-> warehouse

2. Configure application.properties

3. Run

mvn spring-boot:run

4. Swagger

http://localhost:8080/swagger-ui/index.html
